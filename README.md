# intro-to-laravel

## Overview
A simple introduction to Laravel framework with examples. 

## Features
- Basic Laravel structure
- Routing and Controllers
- Blade templating engine examples

## Technologies
- PHP
- Laravel

## How to Use
1. Clone the repository.
2. Install dependencies using `composer install`.
3. Run the application using `php artisan serve`.
